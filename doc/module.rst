Complex safe python module
==========================

To install the complex safe module run,

.. prompt:: bash

    pip install .

.. note::

    This is not needed if you completed the :ref:`complexify_library` install section.


Please refer to the :ref:`complexify_api` and  :ref:`complexify_examples` on how to use the module.
